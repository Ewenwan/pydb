Pydb - Extended Python Debugger

pydb is an expanded version of the Python debugger, pdb.py We follow gdb's command set unless there's good reason. So new or modified command names generally refer to the function they have in gdb. Some of the commands added from gdb:

  • run (restart)
  • frame (Set frame by absolute number)
  • set, show for listsize,
  • info for: args, break, line, locals, globals, source
  • disassemble
  • delete (delete a breakpoint by number)
Many commands that were gdb-like have been expanded:
  • up, down now accept an optional numeric position argument
  • next, step now accept an optional numeric count. Step skips over "def" statements that are not calls
  • clear (Set frame by absolute number)
Also added are line tracing, regression testing (for quality), proper installation as a standalone program. And this just scratches the surface of the improvements.
