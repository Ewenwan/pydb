Pydb – Extended Python Debugger

pydb is an expanded version of the Python debugger loosely based on the gdb command set. It also has all of the features found in an earlier version of pydb.py that was distributed with the debugger GUI ddd.

My latest project has been working on Kent Sibilev's debugger for ruby. There is a work-in-progress guide for the SVN code. Note however this has a number of features that aren't in the prior releases, i.e. version 0.9.3 and earlier. We also have a couple of cool debuggers for GNU Bash and GNU Make.
